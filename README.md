## Intercom Swap Simple (Termux Edition)

A lightweight, real-time peer-to-peer swap simulator with beautiful UI ‚Äî focused on TNK ‚Üî BTC style swaps.  
Built for quick testing on Android via **Termux** ‚Äî no Telegram bot, no heavy frontend framework, just Express + Socket.IO + modern HTML/CSS.

## Features
- Create swap rooms instantly
- Real-time chat between peers in the same swap
- Manual status updates (Waiting ‚Üí Processing ‚Üí Completed/Cancelled)
- Sleek dark-mode UI with glassmorphism, gradients, and neon accents
- Runs entirely on localhost (accessible via browser on the same phone)

## Requirements
- Termux (official from Play Store)
- Node.js & git (installed via pkg)

## Quick Start (in Termux)
